# IT Asset & Patch Tracker

A Java and MySQL application for managing IT assets and tracking software patches.

## Features

- Add, update, delete, and search assets
- Add software patches
- Install patches on assets
- View installed patches
- Generate assets needing updates reports
- Relational database design using MySQL
- JDBC database connectivity

## Technologies Used

- Java
- MySQL
- JDBC
- VS Code
- SQL

## Database Tables

- assets
- patches
- asset_patches
- latest_patches
